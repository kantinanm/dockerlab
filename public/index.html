การพัฒนาและการติดตั้งแอพพลิเคชั่นโดยใช้ซอฟต์แวร์คอนเทนเนอร์

ซอฟต์แวร์คอนเทนเนอร์ในปัจจุบันถูกใช้งานอย่างแพร่หลาย และ Docker คือซอฟต์แวร์คอนเทนเนอร์ที่มีตัวจัดการและได้รับความนิยมอย่างต่อเนื่อง

ในการอบรมครั้งนี้ผู้เข้าร่วมจะเข้าใจถึง วิถีหรือแนวทางการพัฒนาซอฟต์แวร์แบบ Continuous Integration & Continuous Deployment : (CI/CD) ตั้งแต่แรกเริ่มของการพัฒนาจนถึงการติดตั้ง นำเสนอผ่าน Docker เริ่มจากการจัดทำ Docker Image ,การจัดแอพพลิเคชั่นให้อยู่ในรูปแบบสถาปัตยกรรมซอฟต์แวร์คอนเทนเนอร์ การใช้งานซอฟต์แวร์ที่ใช้ควบคุมเวอร์ชั่นซอสโค้ด ร่วมกับระบบ Automation และดำเนินการติดตั้งผ่านซอฟต์แวร์ Jenkins (CI/CD Server)
